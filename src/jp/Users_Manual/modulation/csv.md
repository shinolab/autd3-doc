# Csv

`Csv`はCsvファイルをもとに構成される`Modulation`である.

## Install

### Rust

```shell
cargo add autd3-modulation-audio-file
```

### C++ (CMake)

```ignore,filename=CMakeLists.txt
target_link_libraries(<TARGET> PRIVATE autd3::modulation::audio_file)
```

### C#/Unity/Python

メインライブラリに含まれている.
